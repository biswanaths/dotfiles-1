# Dotfiles
Various .rc files for my command line experience


## Installation
    git clone https://github.com/ajh17/dotfiles.git
    chmod +x ~/dotfiles/bootstrap && ~/dotfiles/bootstrap

### My Environment:
- OS       : Xubuntu 14.04 Trusty Tahr
- Terminal : zsh + tmux
- Editor   : Vim
- IRC      : Weechat
